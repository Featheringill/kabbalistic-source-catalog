# Threshold Press — Kabbalistic Source Catalog

**Publisher:** Threshold Press (*Recovering what was concealed*)
**Translator:** L.L. Featheringill
**Focus:** First complete scholarly English translations of major Lurianic and Sabbatian Kabbalistic texts

This repository contains a public catalog of primary source materials, manuscripts, and scholarly references used in the Threshold Press translation corpus. The translations themselves are not included here.

## How to Use This Catalog

Each subdirectory contains:
- `SOURCE_CATALOG.md` — Primary sources with manuscript references, URLs, and structural outline
- Additional files as noted (manuscript notes, codex contents, etc.)

The master `BIBLIOGRAPHY.md` at root lists all secondary scholarly sources referenced across the corpus.

No translation content is included. Researchers seeking the translations should visit [threshold-press.com](https://threshold-press.com) or search Amazon KDP for "Threshold Press" + the work title.

## Series Note

The five works trace the full arc of Lurianic Kabbalah from its systematic foundation through its most radical departure:

1. **Etz Chayyim** (Vital/Luria) — the foundational system
2. **Sha'ar ha-Kavanot** (Vital) — the liturgical application
3. **Sha'ar ha-Gilgulim** (Vital) — the doctrine of soul-transmigration
4. **Emek HaMelech** (Bachrach/Sarug) — the divergent school (Tehiru Ila'ah dispute)
5. **Derush ha-Tanninim** (Nathan of Gaza) — the Sabbatian theological rupture
